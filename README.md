# VoIP Calling Setup using PBX

## Project Overview
This project demonstrates a basic VoIP communication setup using PBX and softphone.

## What I Did
- Configured PBX server
- Created extensions
- Configured Zoiper softphone using SIP credentials
- Tested calling between systems (Host and VM)
- Troubleshooted audio and connectivity issues

## Technologies Used
- VoIP (SIP, RTP)
- Zoiper
- VirtualBox

## Learning Outcome
- Understanding of VoIP call flow
- Basic networking and troubleshooting skills
